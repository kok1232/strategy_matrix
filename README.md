# 策略矩陣線上工具 (Strategy Matrix Online Tool)

🚀 **專為高階策略管理設計的直覺式矩陣梳理工具**

本專案是一個兼具實用性與便利性的線上工具，旨在幫助 MBA/EMBA 同學、企業經理人及策略規劃者，在進行個案分析或企業策略擬定時，能更流暢、直覺地建構出嚴密的「策略矩陣」架構。

👉 **[立即線上體驗策略矩陣工具](https://kok1232.github.io/strategy_matrix/)**

---

## 💡 專案緣起與靈感

在修習高階策略管理課程期間，**「策略矩陣」** 一直是課堂上討論度最高、也最消耗腦細胞的核心主題。每次面對複雜的商業個案，大家總需要花費大量時間在白板或草稿上反覆梳理各種策略維度與延伸矩陣。

為了向指導教授 **蔣哥** 與 **Betty 老師** 致敬，並感謝兩位老師在課堂上激發的無數靈感，作者決定與 **AI 隊友（AI Agent）** 展開一場跨界協作。希望透過這個工具，將抽象的策略管理理論轉化為落地的數位生產力，為學弟妹及未來的修課同學們提供更便利的學習助力。

---

## 🛠️ 開發幕後與挑戰

別看最終的網頁介面簡潔直覺，在與 AI 協作開發的過程中，我們面臨了不小的挑戰：
* **語意轉譯的困難：** 高階策略管理的邏輯非常宏觀且嚴密（包含核心能力、價值活動、延伸矩陣等），要將這些抽象的商業理論，精準翻譯成 AI 聽得懂的 Prompt（提示詞）並轉化為無 Bug 的程式架構，中間經歷了無數次的邏輯衝突與修正。
* **反覆的指令攻防：** 為了讓互動介面完美契合策略矩陣的思考流程，作者與 AI 來回校正了幾十個版本的 Code，才終於把這套工具「磨」了出來。

---

## ✨ 核心功能與特色

* **直覺式架構導引：** 擺脫傳統紙筆或複雜 Excel 的限制，跟著工具引導快速填入策略元件。
* **響應式網頁設計：** 支援跨裝置瀏覽，不論是用筆電做個案，還是用手機快速檢視都沒問題。
* **開箱即用：** 無需下載任何軟體、無需註冊登入，點開連結即可直接使用。

## 🎯 適用對象與使用情境

如果你符合以下特徵，這個工具將會是你的神隊友：
1. **MBA / EMBA 學生：** 在高階策略管理課程中，需要快速進行個案分析、繪製策略矩陣圖表時。
2. **企業策略規劃者 (Project Manager / Strategy Planner)：** 評估公司內部核心能力與外部價值活動的關聯性時。
3. **對 AI 協作有興趣的開發者：** 參考如何將傳統管理學理論與 AI 程式生成做深度結合。
